# Golden State Getaway 🌞

Interactive itinerary map for the SF (Aug 6–12) + LA (Aug 12–16, 2026) trip.

**Live site:** https://baaurora.github.io/sf-la-trip/

- Full written plan: [ITINERARY.md](ITINERARY.md)
- Click any stop in the sidebar to fly to it on the map
- **✎ Edit** to add / edit / reorder / delete stops (saved in your browser via localStorage)
- **☕ Nearby bites** toggles the unscheduled cafe/snack pins
- **⤓ Export** downloads your edited itinerary as JSON; **↺ Reset** restores the original plan

Built with [Leaflet](https://leafletjs.com/) + CARTO Voyager tiles. No build step — plain HTML/CSS/JS.
